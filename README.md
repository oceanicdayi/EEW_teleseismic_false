# EEW_teleseismic_false

本專案為「115 年 2 月 23 日地震預警系統誤報事件」的公開說明頁面，目的在於透明說明事件經過、原因與改善措施。

## 專案內容

- `index.html`：單頁式說明網站，包含事件時間軸、誤判原因、原始技術資料與後續改善方向。
- 本 README：提供專案背景與使用方式。

## 事件概要

- 2026/02/23 01:01 系統誤發預估規模 4.6、震度 3 級之國家級警報。
- 實際原因為馬來西亞外海 M7.1 遠震，震波被幾何中心法誤判為近海地震。
- 後續改善重點為導入「P 波振幅閾值」機制，以過濾遠震干擾。

## 使用方式

直接以瀏覽器開啟 `index.html` 即可瀏覽，不需額外建置或安裝。

## 語言

主要內容為繁體中文，技術資料區段含必要之英文原始資料。
